# 🚀 KamaClaude 上传 GitHub - 最终指南

## 你需要做的（只需要5分钟）

### 1. 在 GitHub 上创建仓库

1. 打开 https://github.com/new
2. 填写：
   - Repository name: `kamaclaude`
   - Description: `A local AI Agent system, like Claude!`
   - 选择 **Public**（公开开源）
   - ❌ **不要**勾选 Initialize with a README
   - ❌ **不要**勾选 .gitignore
   - ❌ **不要**勾选 License
3. 点击 **Create repository**

---

### 2. 在你的 PowerShell 中执行（请复制完整的代码块）

```powershell
cd "c:\Users\Administrator\Documents\trae_projects\kama claude"

# ===== 配置 Git（只需要第一次用）=====
# 替换成你的名字和邮箱
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

# ===== 提交代码 =====
git add .
git commit -m "Initial commit: KamaClaude - A local AI Agent system"
git branch -M main
```

---

### 3. 连接到 GitHub 并推送（这一步看 GitHub 页面的提示）

GitHub 页面创建完成后，会显示类似这样的命令：
**复制 GitHub 页面上的命令**，不要自己手打！

类似这样：

```powershell
git remote add origin https://github.com/你的用户名/kamaclaude.git
git push -u origin main
```

## 🤔 第一次 push 时提示需要登录？

**现在 GitHub 不能用密码了，需要 Token！**

### 创建 Personal Access Token:

1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 填写：
   - Note: KamaClaude
   - Expiration: 选一个期限
   - Scopes: 勾选 ✅ repo
4. 点击 "Generate token"
5. **复制生成的 token（只显示一次！）**

### 然后 push:
- **Username**: 你的 GitHub 用户名
- **Password**: 粘贴刚才复制的 token
