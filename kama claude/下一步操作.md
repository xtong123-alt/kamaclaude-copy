## 🔧 解决步骤

### 第一步：在 GitHub 上创建仓库（必须先做！）

1. 访问：https://github.com/new
2. 填写信息：
   - **Repository name**: `kamaclaude`
   - **Description**: `A local AI Agent system, like Claude!`
   - **Public/Private**: 选择 **Public**（开源）

3. 点击 **Create repository**

### 第二步：创建完成后，在 PowerShell 中执行

GitHub 创建仓库后，页面会显示命令，复制页面上的命令执行。
